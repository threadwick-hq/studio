# Naming candidates

The working name **stitchgrid** is already taken — there's a cross-stitch
pattern designer published as *StitchGrid* (`stitchgrid.lovable.app`), and the
`Stitch*` space generally is badly overcrowded (Stitch Fiddle, Stitchworks,
Stitch Painter, Google "Stitch", Stitch Data). This is a shortlist of
replacement names for the tool — a WYSIWYG crochet granny‑square chart designer
where you build the chart *in the round*, with live radial symmetry, exactly the
way you'd actually crochet.

## What we're naming for

- **The craft is built around one tool — the hook.** ("Crochet" itself is French
  for *little hook*.)
- **You work in the round.** Concentric rounds grow outward from a magic‑ring
  centre; a finished motif is a small, even, radially‑symmetric round.
- So the strongest names evoke *a small hook* or *a little round / ring / whorl* —
  which is exactly what every name below means.

## How these were vetted

Each finalist was checked with a **bare Google search** (not just a qualified
"X crochet app" search — those miss the small Etsy/Instagram/YouTube/Shopify
brands that actually cause collisions). A name made the list only if a bare
search surfaces **no product, app, shop, or company using it** and **nothing
that would bury it or embarrass it** in results.

> Not yet checked: formal **trademark** registrations (USPTO/EUIPO) and exact
> **domain / app‑store handle** availability. Do those before committing to one.

## Shortlist

### Top tier — cleanest and most usable

**Hooklet**  · *"a small hook"*
The friendliest, cleanest option. The hook is the one tool every crocheter
holds, so the name says "crochet" instantly without being generic. A bare
Google of *Hooklet* returns **only dictionary entries** — no product anywhere.
(Nearest neighbour is *Hookle*, a social‑media app — different word, different
industry.) Easy to say and spell.

**Roundlet**  · *"a little round / small circle"*
You build the chart in the round, ring by ring; a granny square *is* a little
round. A natural sibling to Hooklet (same `-let` diminutive). Bare Google
returns only dictionary + historical‑costume references — no products, apps, or
crochet brands.

**Rondure**  · *"roundness; a circle; graceful curvature"* (Shakespeare, 1609)
The more elegant / premium‑sounding option, evoking the round, symmetric form
the tool produces. Bare Google is dictionary‑dominant; the only notable
commercial user — *Rondure Global*, an investment fund — wound down in 2024
(different domain, now defunct). Clean.

### Second tier — clean and evocative, but more niche

**Annulet**  · *"a little ring"* (in heraldry, a *voided roundel* — literally a
ring/round). Each crocheted round is a ring; the whole chart is concentric
rings. No software/app/crochet use; the only namesakes are in unrelated niches
(a poetry journal, a board game, a small jewellery line) that wouldn't conflict
or compete.

**Verticil**  · *"a whorl — parts arranged in a circle around a single axis"*
Conceptually the most precise match: this is literally the tool's superpower —
stitches arranged with radial symmetry around a centre point. Bare Google
returns only botanical/dictionary results. Trade‑off: it's the most obscure
word here, so the least familiar and hardest to pronounce on sight
(*VER‑ti‑sil*) — highest ownability, lowest recognition.

---

## Suggested pick

- **Hooklet** if you want the friendliest, most obviously‑crochet, cleanest name.
- **Roundlet** if you want to foreground the *in‑the‑round* workflow (pairs
  beautifully with Hooklet as a sibling).
- **Rondure** if you want something more elegant / premium.

---

## Using a descriptor ("Designer", "Studio", "Charts", …)

A name doesn't have to be one coined word — a distinctive word **+ a category
descriptor** is just as ownable, googles cleanly, and says what the tool does.
Descriptor palette: **Designer · Studio · Charts · Workshop · Atelier · Lab**.

**Route 1 — any clean root above + a descriptor** (inherits the vetting, since
the lead word already has zero product collisions):

- **Hooklet Studio** / **Hooklet Designer**
- **Roundlet Studio** / **Roundlet Designer**
- **Rondure Studio** / **Rondure Designer**
- **Annulet Studio** · **Verticil Studio**

**Route 2 — an evocative word the descriptor disambiguates** (verified — no
crochet/design product uses these):

- **Concentric** — *Concentric Crochet* / *Concentric — Crochet Chart Designer.*
  Names the concentric‑rounds structure directly and is distinctive in the craft
  space. (The bare word is used by some unrelated tech firms, so keep "Crochet"
  in the name to disambiguate.)
- **Magic Ring Studio** — instantly recognizable to crocheters; it's how every
  in‑the‑round piece starts. Caveat: "magic ring" is a generic *technique* term,
  so it's weaker for trademark/SEO — the tutorials will outrank you.

Heads‑up on the incumbents this pattern bumps into — *Crochet Studio*, *Stitch
Fiddle*, *Intwined Pattern Studio*, *Stitch Painter* — so the **lead word must
carry the distinctiveness**; the descriptor alone won't.

## Strong names that were rejected (and why)

So these don't get re‑suggested — each *seems* available but fails the bare
search:

| Name | Why it's out |
| --- | --- |
| **Whorl** | Multiple existing apps — a *Whorl* generative‑art app and *Whorl* biometric software. |
| **Roundel** | *Roundel* is Target's retail‑media network — dominates Google. |
| **Roundwork** | A `roundwork` R package (CRAN) **and** a `roundwork.co` video platform. |
| **Hooknest** | `hooknest.com` + a *HookNest* crochet‑patterns site + hardware products. |
| **Hookwork** | `hookwork.shop` + crochet shops — and Googles into "hookworm" (parasite). |
| **Hookwise** | Existing *Hook Wise Crochet* YouTube channel — same niche. |
| **Skeinly / Yarn\*** | Collides with existing *Yarnly*, *YarnBuddy*, *YarnPal*. |
| **Loop\*** (Looplace, Looplet, …) | Saturated: *LoopCraft* ×2, *LooseLoop*, `looplet.app`. |
| **Grannify / Grannygram** | Each is already a shipped app. |
| **Crochetery / Crochette** | Used by many existing crochet shops/handles. |
| **Crochetier** | It's simply the word for *a person who crochets* (+ an existing brand). |
| **Picot / Volute / Gyre / Circlet / Ringlet / Umbel / Tessella** | All taken by existing software or companies. |
| **True Round / Rosette** (+ Designer/Studio) | *TrueRound* is a golf app (`trueround.app`); *Rosette Designer* already exists (a guitar‑rosette design app — direct "…Designer" collision). |
| **Stitch\*** (incl. the current *StitchGrid*) | Extremely crowded — see top of file. |
